{% if not email_configured %}
1

Enable 2-Factor Authentication

  1. Go to Google Account Settings
  2. Click Security in the left sidebar
  3. Find 2-Step Verification and turn it ON
  4. Complete the setup process (phone verification, etc.)
Important

You must have 2-Factor Authentication enabled to generate App Passwords.

2

Generate App Password

  1. Still in Security settings
  2. Click 2-Step Verification
  3. Scroll down to App passwords
  4. Click App passwords
  5. Select:
    • App: Mail
    • Device: Other (custom name)
    • Name: "IGNTU Placement Cell"
  6. Click Generate
  7. Copy the 16-character password (like: abcd efgh ijkl mnop)
3

Update Configuration in app.py

Open your app.py file and find this section:

EMAIL_CONFIG = {
    'smtp_server': 'smtp.gmail.com',
    'smtp_port': 587,
    'email': 'your-email@gmail.com', # üëà Replace this
    'password': 'your-app-password', # üëà Replace this
    'sender_name': 'IGNTU Placement Cell'
}

Replace:

  • your-email@gmail.com ‚Üí Your actual Gmail address
  • your-app-password ‚Üí The 16-character App Password from Step 2
Security Note

Never share your App Password. It gives full access to your email account.

4

Restart & Test

  1. Save the app.py file
  2. Restart your Flask application
  3. Test by generating a student registration link
  4. Check if the email is sent successfully
Testing Tip

Use your own email address first to test if the system is working correctly.

Professional Recommendation

For production use, consider professional email services like SendGrid, Mailgun, or Amazon SES for better deliverability and reliability.

Current Workaround (No Email Setup Needed)

Until you set up email, you can still use the system:

  1. Generate student registration links normally
  2. View OTPs in the Admin OTP Viewer
  3. Share the link and OTP with students manually (WhatsApp, SMS, etc.)
  4. Students can register successfully using the provided OTP
This Works Perfectly!

Many institutions use this manual approach for better control and security.
